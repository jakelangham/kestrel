Welcome to Kestrel's documentation!
===================================

This is the documentation for **Kestrel**, a free Fortran code for simulating
sediment-laden Earth surface flows, such as debris flows, landslides and flash
flooding.

To access the code, or report a problem, consult the `github repository
<https://github.com/jakelangham/kestrel>`_.

.. note::

   This documentation is under active development.
